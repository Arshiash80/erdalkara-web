# Photo assets

The design references these local images. Drop the real high-res photos here
with these exact filenames and they'll appear automatically (the design shows
dark gradient placeholders until then):

- `tools-roll.jpg`   — barber tools / equipment (used in Story + Gallery)
- `interior-1.jpg`   — salon interior (Story inset)
- `interior-2.jpg`   — salon interior wide (Gallery hero frame)
- `storefront-1.jpg` — storefront / exterior (Gallery wide frame)
- `certificates.jpg` — framed certificates wall (Gallery frame)

Recommended: high resolution, well-lit, horizontal where noted. The site
applies a subtle dark/contrast filter to keep the cinematic mood consistent.

A few gallery frames currently use temporary Unsplash images — swap those for
real work photos when available (see `app/_design/markup.ts`, `data-lb` frames).
